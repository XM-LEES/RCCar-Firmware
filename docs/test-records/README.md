# 测试记录目录

本目录保存 `RCCar-new/` 可提交、可追溯的 Markdown 测试记录。

适合放入本目录的内容：

- Ubuntu 静态检查记录。
- Windows / Keil 编译、下载和烧录摘要。
- 台架 UART 解析检查。
- 架空轮测试。
- 低速落地测试。

原始日志、串口抓包、Keil 编译输出、终端转写和大附件先放入 `../../logs/`。需要纳入版本管理时，在本目录写 Markdown 摘要并链接对应原始资料。

推荐文件名格式：

```text
YYYY-MM-DD-<scope>-<result>.md
```

运动相关记录必须包含：

- 仓库分支和 commit。
- 固件构建命令或静态检查命令。
- 测试场景。
- 是否架空。
- 命令输入。
- 实际输出。
- RC 接管、刹车、急停、通信超时状态。
- 原始日志路径，如有。
- 未覆盖风险。
